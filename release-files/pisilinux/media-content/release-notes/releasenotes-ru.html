Команда Anka с гордостью представляет Pisi Linux 2.0
Pisi Linux - это операционная система с обновлением 'роллинг-релиз'.
Она была создана как наследник основанного на PiSi дистрибутива Pardus Linux после объявления о конце разработки Pardus в 2012-м году.

Pisi Linux содержит несколько уникальных технологий:
COMAR(COnfigurationMAnageR), управляет и настраивает установленное ПО.
MUDUR, разработанный для управления процессом загрузки.
PISI(Packages Installed Successfully, as Intended), Пакетный менеджер разработанный для Pardus Linux.
YALI (Yet Another Linux Installer), собственный установщик системы.
KAPTAN, программа, которая автоматически стартует после первой загрузки для того, чтобы помочь с настройкой системы.

Системные Требования
64-битный процессор
Минимум 2 Гб ОЗУ и 12 Гб дискового пространства
Рекомендуется создать раздел подкачки для Pisi Linux (это можно сделать во время установки)

Мультимедиа
Pisi Linux сейчас представлен только 64-битной версиейю Однако, 32-битные пакеты включены в систему для таких приложений, как
Skype и Steam. Кодеки для проигрывания аудио и видео файлов тоже включены.
